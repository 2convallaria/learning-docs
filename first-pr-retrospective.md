## 2. 我学到了什么（流程方面）

2. 关键步骤2：在自己的仓库中找到对应的副本并且进行编辑
3. 关键步骤3：提交修改，填写修改的主题和对应的描述
4. 关键步骤4：选择创建一个新的分支（没有权限），在新分支中修改提交pr
